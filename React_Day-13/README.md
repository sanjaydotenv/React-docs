# 🛒 React Mini E-Commerce Project

A simple **React Shopping Cart** project built to practice **React Context API**, **API fetching**, and **state management**.

## 🚀 About the Project

In this project, I used the **React Context API** to share data between components without using props drilling. All product data is fetched from the **Fake Store API**, and I used **Axios** to handle the API requests.

This project helped me understand how a basic shopping cart works in React and how to manage global state efficiently.

## ✨ Features

* 📦 Fetch products from Fake Store API
* 🌐 API requests using Axios
* 🛒 Add products to the cart
* ➕ Increase product quantity
* ➖ Decrease product quantity (minimum quantity is **1**)
* ❌ Remove products from the cart
* 💰 Automatically update the total price based on quantity
* 🎨 Clean and responsive UI with Tailwind CSS

## 🛠️ Technologies Used

* ⚛️ React
* 🔄 Context API
* 🌐 Axios
* 🎨 Tailwind CSS
* 🪝 React Hooks (`useState`, `useEffect`, `useContext`)

## 📚 What I Learned

* Using **Context API** for global state management
* Fetching data from an external API with **Axios**
* Managing shopping cart functionality
* Updating UI based on state changes
* Building reusable React components
