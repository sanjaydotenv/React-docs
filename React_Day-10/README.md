# React User Management CRUD

A simple **React CRUD (Create, Read, Update, Delete)** application built using **React Hook Form**, **Tailwind CSS**, and **LocalStorage**. This project allows users to create, update, delete, and view user cards with form validation.

## 🚀 Features

- Create new users
- Update existing users
- Delete users
- Form validation using React Hook Form
- Data persistence with LocalStorage
- Responsive UI with Tailwind CSS
- Dynamic user cards

## 🛠️ Technologies Used

- React.js
- React Hook Form
- Tailwind CSS
- Nano ID
- LocalStorage

## 📂 Project Features

- Add User Form
- User Card UI
- Update User Details
- Delete User
- Responsive Design
- Client-side Data Storage
